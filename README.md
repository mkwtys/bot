# hubot

hubot for slack

## config

```sh
# for hubot
export HUBOT_NAME='...'
export HUBOT_URL='...'

# for hubot-slack
export HUBOT_SLACK_BOTNAME='...'
export HUBOT_SLACK_TEAM='...'
export HUBOT_SLACK_TOKEN='...'

# for hubot-heroku-keepalive
export HUBOT_HEROKU_KEEPALIVE_URL='...'
export HUBOT_HEROKU_SLEEP_TIME='...'
export HUBOT_HEROKU_WAKEUP_TIME='...'

# for hubot-docomo-dialogue
export HUBOT_DOCOMO_DIALOGUE_API_KEY='...'
export HUBOT_DOCOMO_DIALOGUE_P='...'
```
